# Sparklers Dashboard
Scripts to get data about Sparklers from WordPress and other sites.
Upload that data to Google Sheets.

## Components

- **WordPress** – Login data
- **WooCommerce** – Subscription info
- **Wishlist** – Points and gold stars
- **Upload-to-Sheets** – Upload CSVs to Google Sheets
- **Klaviyo** – Email campaign recipients, opens, and clicks
